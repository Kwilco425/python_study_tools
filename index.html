✨ WHAT MAKES THESE DIFFERENT

  • Memory tricks for every question. Not just an explanation — a phrase or mnemonic to help the concept actually stick.
  • Mistakes are first-class. Get one wrong? You see your answer in red, the correct answer in green, with a plain-English explanation and the memory trick.
  • Topic-level feedback. The final report breaks results down by topic and generates a targeted study guide based on what you actually missed.
  • Halfway check-in at question 15 so you can pace yourself.
  • Code editor for code questions (quiz 01) — write actual Python, get graded against real test cases. The grader translates Python to JS at runtime, no Python runtime needed in the browser.
  • Zero dependencies. One HTML file per quiz. No build step, no CDN, no internet required after download. Run them anywhere.
  • Mobile responsive — study on your phone if you want.
